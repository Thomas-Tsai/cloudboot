開機步驟：
  1. 開機媒介選擇：
    1. 光碟開機
      2. 下載 ipxe.dsk 並以指令 "cat bin/ipxe.dsk > /dev/fd0" 安裝ipxe.dsk到軟機片中。
    2. USB碟開機
      3. 下載 ipxe.usb 並以指令 "cat bin/ipxe.usb > /dev/sdX" 安裝ipxe.usb到USB碟中。
    3. 軟碟開機
      4. 下載 ipxe.iso 並燒錄到光碟機中，就完成ipxe的安裝了。
  2. 選擇雲端應用程式，目前支援的有：
    1. DRBL Live
    2. Clonezilla Live
    3. GParted Live
    4. Netinstall Linux
    5. Freedos
    6. Memtest
  3. 自動下載與啟動
